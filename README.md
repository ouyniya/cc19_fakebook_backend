# FAKEBOOK Workshop



## Step 1 
### create file `server.js` 

### start project
```bash
npm init -y
```

### update `package.json`
```js
"scripts": {
    "dev": "nodemon server.js",
    "start": "nodemon server.js"
  },
```

### create `.gitignore` file
```js
node_modules/
.env
```

### installation
```bash
npm i 
```

### git init
```bash
git init 
```
